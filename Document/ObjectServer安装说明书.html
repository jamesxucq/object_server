Object Server安装说明书

 

一.Debain包安装

    从网址https://github.com/jamesxucq/object_server/地址下载

object-server_0.6.1_amd64.deb, ObjectServer的Debain安装包.

安装测试环境 ubuntu-24.04-live-server-amd64.

下载安装包，并执行以下命令安装：

    sudo dpkg -i object-server_0.6.1_amd64.deb

ObjectServer的安装目录为 /opt/object_server/

    sudo apt-get install -f

如果有依赖问题,执行上面的命令修复依赖.

   sudo apt-get remove object-server

执行上面的命令卸载ObjectServer.

 

二.编译安装

    源码仓这个目录NetBeansProjects下载本软件的源代码.

    软件的用的IDE netbeans64自带的软件构建系统，如想自行编译安装本软件，

请安装netbeans64.
